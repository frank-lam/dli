# 删除数据库<a name="dli_02_0030"></a>

## 功能介绍<a name="sa13cbeda85f34f68b7ffe23a6b3694be"></a>

该API用于删除空数据库，若待删除的数据库中存在表，则需先删除其中的所有表。删除表API请参见[删除表](删除表.md)。

    DELETE /v1.0/\{project\_id\}/databases/\{database\_name\}
